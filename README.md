# Aletheia Library Management System

<h2>App name:</h2>
Aletheia (from the Greek word for “truth” or “unconcealed knowledge.” suggest that books reveal wisdom)

<h2>Description:</h2>
An app made to track for users to find the book that they want and to check if it is available within the application. Users will also be able to keep track of books that they borrowed and its return schedule.

<h2>Target audience:</h2> 

<h2>Goal:</h2> 
The goal of this app is to check and track the availability of the book if it’s possible to be borrowed and object driven focus on books

<h2>Problem being solved:</h2>
To make the target audience’s life easier by making an online app that tracks the availability of the books and borrow/reserve them without the need to go to the library directly.

<h2>Competition:</h2>
(Koha, Worldcat) Our app is unique because it integrates school email verification and multi-factor authentication while providing real-time book availability, borrowing tracking, and return scheduling in one centralized platform.
